## Usage

1. Install is used to install all dependencies for a project.

    ```bash
    $ yarn install
    ```

2. Write interview data
    
    in `index.js` file
    ```js
    const datas = [
        {
            'question': 'please write question!!',
            'tags': ['tag1', 'tag2'],
            'answer': 'please write answer!!'
        },
    ];
    ```

3. Run program
   
    ```bash
    $ yarn start
    ```
    or

    ```bash
    $ node index.js
    ```
